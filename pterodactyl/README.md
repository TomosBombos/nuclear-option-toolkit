# Pterodactyl install &nbsp;·&nbsp; ✅ working

Hosting your *Nuclear Option* server on a **Pterodactyl panel**? This is your version.

### **[⬇ Download the Pterodactyl bundle](https://github.com/TomosBombos/nuclear-option-toolkit/releases/latest/download/nuclear-option-toolkit-pterodactyl.zip)**

The download is a ready-to-go folder with **everything inside** — BepInEx, the NukeStats plugin,
all 18 missions, the bot, and the web command centre.

1. Unzip it anywhere on your PC.
2. You need **Python 3.8+** ([python.org](https://www.python.org/downloads/) — tick *Add to PATH*).
3. Run **`install.bat`** (Windows) or **`./install.sh`** (macOS/Linux).
4. Enter your panel's SFTP + API details — **every field explains what it is and where to find it**.
5. It pushes the plugin, missions and config to your server over SFTP and makes it boot **modded
   with no panel edits**, then launches the bot + your dashboard.

Full step-by-step + troubleshooting is in the `README.md` inside the download.
The toolkit source code lives in [`../src`](../src).
