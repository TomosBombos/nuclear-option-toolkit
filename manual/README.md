# Manual install (host it by hand) &nbsp;·&nbsp; ⚠️ beta (lightly tested)

For people who run the dedicated server **themselves** and want to add the toolkit by hand. Ships
**everything** — both BepInEx packs (Windows + Linux), the plugin, all 18 missions, the bot, and
the web command centre — plus clear drag-and-drop instructions.

### **[⬇ Download the Manual bundle](https://github.com/TomosBombos/nuclear-option-toolkit/releases/latest/download/nuclear-option-toolkit-manual.zip)**

1. Unzip it.
2. Follow the `README.md` inside to drag the game-side files into place and paste the startup line.
3. Run the small installer to write your config (each field explains what it is and where to find it),
   then start the bot + web command centre.

> ⚠️ This path is built but only lightly tested so far — please
> [open an issue](https://github.com/TomosBombos/nuclear-option-toolkit/issues) with anything you
> hit. The **Pterodactyl** version is the tested one.
