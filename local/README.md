# Local install (your own PC) &nbsp;·&nbsp; ⚠️ beta (lightly tested)

Run the whole community server on **your own Windows/Linux PC** — the dedicated server, the
plugin, all 18 missions, the bot, and the web command centre, launched together.

### **[⬇ Download the Local bundle](https://github.com/TomosBombos/nuclear-option-toolkit/releases/latest/download/nuclear-option-toolkit-local.zip)**

1. Unzip it somewhere short (avoid OneDrive / Program Files).
2. You need **Python 3.8+** ([python.org](https://www.python.org/downloads/) — tick *Add to PATH*).
3. Run **`install.bat`** (Windows) or **`./install.sh`** (Linux).
4. It installs the dedicated server with SteamCMD, copies the toolkit in, and launches
   server + bot + dashboard together.

> ⚠️ This path is built but only lightly tested so far — it should work, but expect rough edges
> and please [open an issue](https://github.com/TomosBombos/nuclear-option-toolkit/issues) with
> anything you hit. The **Pterodactyl** version is the tested one.

Full instructions are in the `README.md` inside the download. Source code: [`../src`](../src).
